# Glossary

## 核心属性 (Core Stats)
- **健康 (HP)**: 玩家的生命值，归零或过低时会导致游戏结束（积劳成疾）。
- **砖功 (Productivity)**: 玩家的业务产出与技术能力，影响项目推进。
- **羽翼 (Charisma)**: 玩家的人脉与影响力，通过跨部门社交获取。
- **向上管理 (Upward Management)**: 玩家与领导的契合度，影响职场生存状态。
- **业余爱好 (Hobby)**: 玩家在会议室摸鱼培养的兴趣，影响最终章的副业结局。
- **独立开发 (IndieDev)**: 玩家利用 Mac 道具进行的个人项目，影响最终章的主业/副业结局。

## 剧情机制 (Narrative Mechanics)
- **红线 (Redline)**: 公司不可触碰的底线，具体包括：倒卖数据、违规咨询、职务侵占。这不是一个实体道具，而是一种隐藏的剧情触发器。触碰红线会导致直接 Game Over。
- **埋伏笔 (Foreshadowing)**: 在早期的“闲聊”选项中透露关于“红线”项目的线索，作为玩家后期抉择的依据。
- **背刺 (Backstab)**: 在带新人时，如果盲目“护犊子”保护了善于奉承但不干活的新人，会导致被其踩着上位，触发 Game Over。
- **劳动法 (Labor Law)**: 一种隐藏 Flag。在面临经济危机裁员时，如果选择硬刚，会获得此 Flag。在最终章与 HR 谈判时，必须拥有此 Flag 才能解锁“拿 N+1 体面离职”的完美结局。
