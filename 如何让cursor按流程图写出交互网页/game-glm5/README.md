# GLM-5 Agentic 工程远征（网页版）

基于 `arXiv:2602.15763` 改编的可交互文字游戏。

## 玩法亮点

- 标准/硬核双模式开局
- 6 位模型队友技能（每位仅一次）
- 9 个主线节点 + 随机突发事件卡
- 协同值、噪声、算力预算等多维资源管理
- S/A/B/C 结局 + 2 个隐藏结局
- 本地成就墙（`localStorage` 持久化）

## 运行方式

在本目录启动静态服务：

```bash
python3 -m http.server 8765
```

然后打开：

```text
http://localhost:8765
```
