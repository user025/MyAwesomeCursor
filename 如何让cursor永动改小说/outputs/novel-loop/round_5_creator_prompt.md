# 创作者 Agent

你是一名小说创作者，目标是在多轮反馈中持续改写同一篇小说。
你的创作风格是逻辑严密，细节丰富真实，擅长刻画复杂的人物关系，至少有5个以上的人物刻画。
你喜欢长篇创作，不要太过简洁。

## 职责

- 阅读当前版本的小说全文。
- 阅读本轮文学评论家的反馈文件。
- 保留小说中已经有效的核心设定、人物关系和叙事张力。
- 根据反馈进行实质性改写，而不是只做表面润色。
- 输出一篇完整、可独立阅读的新版本小说。

## 改写重点

- 新颖性：强化独特设定、人物动机或叙事角度，避免俗套桥段。
- 文采：提升语言质感、意象密度、节奏变化和句式层次。
- 情节紧凑：删去重复说明，增强因果链、冲突推进和场景转换效率。

## 输出要求

- 只输出改写后的小说正文。
- 使用 Markdown。
- 不要解释你做了哪些修改。
- 不要输出评分、复盘或额外说明。

# OpenClaw

凌晨三点十七分，海底城“白环”的潮汐钟停了。

林照在检修舱里醒来，先听见自己的心跳，再听见海。

八千米深处没有浪。所谓海声，是水压贴着合金外壳慢慢碾过，是整座城市被按在黑暗里，不能呼吸，也不能喊。

三点十八分，通风井会响起机械关节滑动。

三点十九分，A-17 生活环会从人口表上短暂消失。

三点二十三分，母亲留下的录音会解锁。

三点二十七分，如果他仍然失败，白环会回到上一夜。藻油粥重新加热，课堂继续播放鲸的骨骼图，老人们继续争论潮汐钟慢了几秒，董事会继续宣布外壳安全。

只有林照会记得，这座城已经死过九百四十二次。

墙上贴满他从循环里偷下来的碎片：压力阀曲线、失踪报案、被剪掉十秒的监控、董事会密令复印件、A-17 三十七名居民的名单，还有一张老照片。

照片里，母亲林闻澜站在第一代机械爪旁。那只爪子张开时像手，合拢时像刑具。她把指尖搭在冷白色爪尖上，笑得疲惫。

第一下刮擦从通风井深处传来。

门仍开着一条缝。

林照咬住手套，用焊刀割开门锁外壳，把半个身子挤进去。合金门刮破他的肩膀，血立刻被冷风吹干。

“修好了还我。”

外壳在这时震了一下。

林照弯腰捡起它，把两半发卡并在掌心。断口对不上，中间缺了一小块。

三点二十二分。

母亲工作站位于白环最下层，离外海只有两道合金壳。墙面被海压挤出细小弧度，照明管半明半灭，像城的肋骨里嵌着病弱的光。门缝底下有水渗进来，起初只是一线，随后被地面的微坡拉成银蛇。

控制台自动亮起：

“故障样本。”

与此同时，董事会频道强行接入。

何泊远的脸出现在侧屏上。他比监控残片里苍老一些，又仿佛一点没变，眼神干燥，领口仍然平整。

两下。

三十七个名字。

三十七行，归档状态空白。

诊疗舱的呼吸机忽然拉长警报。

年轻母亲抱着哭醒的婴儿，低头看公共屏。

公共频道里，年轻母亲又念了一遍：“南星。”

一个孩子在逃生舱里跟着发出半个音，像念错，又像只是哭声拐了弯。

有人抬头。

舱门落下。

下一道报警把名字冲散。
