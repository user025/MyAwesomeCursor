# 雾城循环

凌晨三点十七分，林照又一次在同一声汽笛里醒来。

窗外的城市被雾裹住，钟楼停在三点十八分以前，街角卖花的老人正把一束白色鸢尾插进空瓶。林照知道，三分钟后会有一辆没有车牌的黑色电车驶过，带走所有听见铃声的人。再过七分钟，城市会重置，所有人忘记刚才发生过什么，除了他。

他已经在这个凌晨里活了九百四十二次。

起初，他以为自己被困在时间里。后来他发现，真正被困住的是这座城市。每一次循环结束，雾都会更低一点，像某种正在合拢的眼皮。林照把线索写满墙壁：钟楼、鸢尾、黑色电车、失踪名单，还有母亲留下的最后一句话。

“不要救我，救那个写下结局的人。”

第九百四十三次醒来时，林照发现墙上多了一行不是自己写的字：

“你终于读到这里了。”
